# J-Tag

Painel familiar aberto por NFC. A primeira versao e uma PWA simples em Next.js para testar o fluxo:

1. Encostar o celular em uma etiqueta NFC.
2. Abrir a URL do app.
3. Selecionar um morador.
4. Entrar com PIN.
5. Ver lembretes, aniversarios e emergencia.

## Rodando localmente

```bash
npm install
npm run dev
```

Abra `http://localhost:3000`.

## Deploy

O projeto foi pensado para Vercel. Depois de conectar o repositorio, a URL gerada pela Vercel pode ser gravada nas etiquetas NFC.
